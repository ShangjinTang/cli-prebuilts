# cli-prebuilts

Recommended CLI prebuilt binaries which are up-to-date (checked everyday).

## Installation

```bash
git clone https://github.com/ShangjinTang/cli-prebuilts.git ~/.prebuilts --depth=1
```

Then add `~/.prebuilts/bin` to your $PATH.
