# cli-prebuilts

Recommended CLI prebuilt binaries which are up-to-date (checked everday).

## Installation

```bash
git clone https://github.com/ShangjinTang/cli-prebuilts.git ~/.prebuilts --depth=1
```

And add `~/.prebuilts/bin` to your $PATH.
